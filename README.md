# Gin + Gorilla 聊天室

- 項目立於 2019.8.30

- 由于 Gin 提供了表单验证的一些特性，我在Controller中就完成了表单的验证工作，这可能看起来有些诡异。

- 为了节省资源，这个项目用到了大量的单例模式（连连接池都是单例的...）
- 在 WebSocket 模块中支持定时的任务列队
